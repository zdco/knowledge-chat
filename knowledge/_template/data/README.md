# 数据文件目录

本目录存放该知识域的所有资料。AI 搜索时会自动包含此目录。

请将相关的源码、文档、配置等文件按类型建子目录存放，例如：

```
data/
├── src/       # 源码文件（*.cpp, *.h 等）
├── jce/       # 接口定义（*.jce）
├── doc/       # 文档（*.md, *.txt）
├── conf/      # 配置文件（*.conf, *.xml, *.yaml）
└── ...        # 其他资料
```

整个知识域目录是自包含的，所有资料都放在这里，方便直接拷贝到部署服务器使用。

注意：二进制文件（图片、PDF 等）不会被搜索工具处理。
